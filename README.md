# 微信表情消消乐 - 真实堆叠版

基于微信表情包的消消乐游戏，采用真实堆叠布局和麻将式反面牌机制。

## 🎮 游戏特色
- **真实堆叠效果**: 不规则偏移，下层牌部分露出
- **策略性玩法**: 可以预判下面的牌，制定移牌策略  
- **完整功能**: 6×10布局，180张牌，10分钟倒计时
- **精美UI**: 渐变背景，毛玻璃效果，动画交互

## 🚀 快速开始
1. 克隆仓库
2. 打开 `index.html` 即可游玩
3. 按照 `assets/README.md` 替换为真实微信表情包
